*   **To**: HQ Agent
*   **Acoustic/Sound Quality Objective (What)**: Restrict all file upload endpoints across the ecosystem (Noble, Titan, Vanguard, Sentinel) to accept exclusively raw, uncompressed `.wav` file formats.
